# Docker-compose

---

- Встановлено Docker та Docker Compose на локальну систему
- Створено багатоконтейнерний застосунок за допомогою Docker Compose
- Налаштовано вебсервер nginx, базу даних PostgreSQL та кеш Redis
- Створено та підключено томи для збереження даних
- Налаштовано спільну Docker-мережу для взаємодії сервісів
- Перевірено роботу вебсервера через браузер
- Виконано підключення до бази даних PostgreSQL всередині контейнера
- Здійснено масштабування вебсервера(додано лоадбалнсер)
- Перевірено стан та коректну роботу запущених контейнерів
