# n8n Automation Workflow

## Мета роботи
Реалізувати автоматизацію обробки заявок з Google Forms та надсилання повідомлень у Telegram за допомогою n8n.

---

## Кроки виконання

### 1. Розгортання n8n
Було розгорнуто локальну інсталяцію n8n за допомогою Docker Compose з використанням PostgreSQL для збереження даних workflow.


---

### 2. Створення Google Form
Було створено форму з такими полями:

- Ім’я
- Email
- Тип запиту
- Опис проблеми
- Пріоритет (Low / Medium / High)

Відповіді з форми передаються у workflow n8n.

---

### 3. Створення Telegram Bot
Було створено Telegram бота через **BotFather**.

Отримано:
- **Bot Token**
- **chat_id**

Бот використовується для надсилання повідомлень про нові заявки.

---

### 4. Побудова n8n Workflow

Було створено workflow з такими кроками:

1. **Webhook Trigger** – отримує дані з форми
2. **Extract Fields** – обробка отриманих даних
3. **PostgreSQL Check** – перевірка дублю по email
4. **Insert Request** – збереження заявки у базу
5. **Telegram Node** – відправка повідомлення (Retry on Fail)
6. **Update Status** – оновлення статусу заявки
