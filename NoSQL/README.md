# Робота з MongoDB — База даних спортзалу

---

## Опис виконаної роботи

### 1. Створення бази даних та колекцій
- Створено базу даних **gymDatabase**
- Створено колекції:
    - `clients`
    - `memberships`
    - `workouts`
    - `trainers`

---

### 2. Структура документів
- **Clients:** `client_id`, `name`, `age`, `email`
- **Memberships:** `membership_id`, `client_id`, `start_date`, `end_date`, `type`
- **Workouts:** `workout_id`, `description`, `difficulty`
- **Trainers:** `trainer_id`, `name`, `specialization`

---

### 3. Заповнення даними
До кожної колекції додано кілька записів з реалістичними даними для імітації роботи реального спортзалу.

---

### 4. Виконані запити
- Отримано список клієнтів віком понад 30 років
- Отримано перелік тренувань із середньою складністю
- Отримано інформацію про членство клієнта за заданим `client_id`

---

## Результат
- Створена та заповнена база даних **gymDatabase**
- Реалізовані базові запити до MongoDB
- Продемонстровано роботу з колекціями та пошуком даних у MongoDB
