# Ansible Automation Project

---

## Створені ролі

### baseline
Базова конфігурація серверів:
- Налаштування SSH-ключів
- Встановлення базових пакетів (vim, git, mc)
- Підготовка системи до подальшої конфігурації

---

### firewall
Налаштування мережевої безпеки:
- Встановлення та запуск firewalld (Amazon Linux)
- Дозвіл доступу до SSH (22)
- Дозвіл доступу до HTTP (80)
- Блокування іншого вхідного трафіку

---

### nginx
Розгортання веб-сервера:
- Встановлення Nginx
- Розгортання конфігурації через шаблон
- Розгортання index.html через Jinja2
- Перезапуск сервісу через handler

---

## Використані Playbook-и

### site.yml
Повна конфігурація сервера:
- baseline
- firewall
- nginx

### baseline-only.yml
Тільки базова конфігурація.

### firewall-only.yml
Тільки налаштування firewall.

### nginx-only.yml
Тільки встановлення та конфігурація Nginx.

---

## Dynamic Inventory

Використано AWS dynamic inventory (`amazon.aws.aws_ec2`), що дозволяє автоматично отримувати список EC2 інстансів за тегами.

---

## Ansible Vault

Для зберігання конфіденційних даних використано Ansible Vault.  
Vault застосовується під час запуску playbook.
