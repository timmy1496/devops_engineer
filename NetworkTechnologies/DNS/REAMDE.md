# Конфігурація DNS

## 1. Поясніть процес резолюції доменних імен.

Процес резолюції доменного імені починається з DNS-запиту клієнта до DNS-сервера. 
DNS-сервер перевіряє локальні записи та, у разі знаходження відповідності, повертає IP-адресу клієнту. 
Якщо запис відсутній, сервер може перенаправити запит до вищих DNS-серверів. 
Отримавши IP-адресу, клієнт використовує її для встановлення мережевого зʼєднання.
